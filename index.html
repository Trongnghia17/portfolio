Dự án

Hệ thống Quản lý Thú Cưng

Xem Demo
Vue.js Vuetify Laravel MySQL Docker REST API

Hệ thống là một nền tảng bán hàng và quản lý cửa hàng thú cưng giúp chủ shop thú cưng dễ dàng quản lý sản phẩm, đơn hàng, khách hàng và thanh toán trực tuyến. Hệ thống hỗ trợ nhiều vai trò (Admin, Khách hàng), tích hợp API thanh toán và báo cáo doanh thu chi tiết.

Các tính năng chính:

  • Quản lý sản phẩm: Thêm, sửa, xóa sản phẩm (thức ăn, phụ kiện, dịch vụ chăm sóc thú cưng).
  • Quản lý đơn hàng: Đặt hàng, cập nhật trạng thái giao hàng.
  • Hệ thống giỏ hàng: Người dùng có thể thêm sản phẩm vào giỏ, áp dụng mã giảm giá.
  • Thanh toán trực tuyến: Tích hợp cổng thanh toán (Stripe, PayPal).
  • Bảo mật: Xác thực JWT Authentication, bảo vệ API với middleware trong Laravel.

Vai trò của tôi trong dự án:

  • Xây dựng giao diện frontend bằng Vue.js và Vuex, tối ưu UX/UI.
  • Thiết kế và phát triển backend bằng Laravel, tạo API RESTful cho hệ thống.
  • Tích hợp cổng thanh toán và xử lý giao dịch an toàn, chống gian lận.
  • Thiết kế cơ sở dữ liệu MySQL, tối ưu truy vấn để giảm tải cho hệ thống.
  • Triển khai hệ thống bằng Docker, giúp dễ dàng mở rộng và quản lý.
